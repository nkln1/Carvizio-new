Iată un prompt actualizat pentru asistentul AI din Replit, specific pentru implementarea în pagina de profil din Service-dashboard, în tab-ul account:

```
Creează o componentă React cu TypeScript pentru secțiunea "Preferințe notificări" care va fi integrată în tab-ul Account din Service-dashboard al proiectului Carvizio. Proiectul folosește React și TypeScript.

Componenta trebuie să fie integrată în pagina existentă de profil service și să conțină:

1. O secțiune "Preferințe notificări" cu două tab-uri sau accordion-uri care să se potrivească cu designul existent al paginii de Account.

2. Prima secțiune: "Notificări email" care să includă:
   - Un toggle master pentru activarea/dezactivarea tuturor notificărilor email
   - Toggle-uri individuale pentru următoarele tipuri de notificări specifice service-urilor:
     * Cereri noi primite
     * Oferte acceptate de clienți
     * Mesaje noi primite
     * Recenzii noi primite
   - Toggle-urile individuale trebuie să fie dezactivate când toggle-ul master este dezactivat

3. A doua secțiune: "Notificări browser" care să includă:
   - Un toggle master pentru activarea/dezactivarea tuturor notificărilor browser
   - Toggle-uri individuale pentru aceleași tipuri de notificări ca la email
   - Un buton "Solicită permisiunea browser-ului" care să apară doar dacă permisiunea nu a fost acordată încă

4. Integrare cu API pentru salvarea și încărcarea preferințelor:
   - Endpoint-uri de API pentru GET și UPDATE ale preferințelor de notificare
   - Gestionarea stării de încărcare și erori

5. Stilizare și design care să se integreze perfect cu restul tab-ului Account din Service-dashboard

Folosește React hooks (useState, useEffect, etc.) și react-query pentru comunicarea cu API-ul. Asigură-te că codul este optimizat și se potrivește cu stilul și funcționalitatea existentă din Service-dashboard.

Adaugă comentarii pentru a explica pașii de implementare și integrare cu restul codului.
```

Acest prompt este specific pentru implementarea în tab-ul Account din Service-dashboard și include toate cerințele pentru secțiunea de preferințe notificări cu ambele tipuri de notificări (email și browser).
