# Prompt pentru Agentul AI Replit: Sistem Simplificat de Cookie-uri Esențiale pentru Site React

Vreau să implementezi un sistem simplu de gestionare a cookie-urilor pentru site-ul meu React cu hosting pe VPS, care să respecte reglementările GDPR dar să folosească doar cookie-uri esențiale (fără marketing și fără vânzarea datelor). Sistemul trebuie să includă:

## 1. Banner de informare pentru cookie-uri

Creează un banner simplu care:
- Informează utilizatorii că site-ul folosește doar cookie-uri esențiale pentru funcționare
- Explică pe scurt ce sunt cookie-urile esențiale și de ce sunt necesare
- Oferă un link către o pagină cu politica de cookie-uri
- Permite închiderea banner-ului cu un buton "Am înțeles"
- Se afișează o singură dată per utilizator (folosind un cookie de consimțământ)

## 2. Implementare a cookie-urilor esențiale în React

- Funcții pentru setarea și citirea doar a cookie-urilor strict necesare (sesiune, preferințe de bază)
- Exemplu de utilizare pentru:
  - Reținerea preferinței de limbă/temă
  - Cookie-uri de sesiune (dacă este necesar)
  - Funcționalități de bază care necesită persistență

## 3. Stocare minimă a datelor

- Implementare simplă folosind localStorage sau sessionStorage unde este posibil
- Dacă e necesar un backend, include doar un endpoint simplu pentru stocarea preferințelor de bază

## 4. Politica de cookie-uri simplificată

- Template pentru o politică simplă care menționează doar cookie-urile esențiale
- Explicație clară că site-ul nu folosește cookie-uri de marketing și nu vinde date

## 5. Integrare în site-ul React

- Componente React pentru banner și gestionarea cookie-urilor
- Cod ce poate fi integrat ușor în aplicația existentă
- Instrucțiuni de implementare pas cu pas

## Detalii tehnice

- Frontend: React (cu hooks)
- Pachetele recomandate: js-cookie sau o soluție similară ușoară
- Fără dependențe complexe pentru marketing/tracking

## Rezultate așteptate

- Cod simplu și concis pentru banner și gestionarea cookie-urilor esențiale
- Politică de cookie-uri simplificată
- Integrare ușoară cu site-ul existent
- Conformitate GDPR minimă necesară pentru cookie-uri esențiale

Te rog să creezi o soluție eficientă și minimalistă care să îndeplinească doar cerințele legale de bază, fără funcționalități suplimentare pentru colectarea sau procesarea datelor în scop de marketing.
